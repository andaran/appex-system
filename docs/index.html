checkRequestLimit

Проверяет, сколько раз пользователь делал запрос за эту минуту.

checkRequestLimit
Parameters
data (object) что пришло с клиента
Returns
Promise: реджектит ошибку в случае привышения лимита

updateState

Обновляет обьект состояния.

updateState
Parameters
data (object) что пришло с клиента
Returns
Promise: реджектит обьект с ошибкой или резолвит обновленное состояние
Example
updateState({
   roomId: "123",
   roomPass: "123",
   params: {
     status: true
   }
});

changeNumericProperty

Прибавляет переданное число к указанному св-ву обьекта состояния.

changeNumericProperty
Parameters
data (object) что пришло с клиента
Returns
Promise: реджектит обьект с ошибкой или резолвит обновленное состояние
Example
changeNumericProperty({
   roomId: "123",
   roomPass: "123",
   property: "status",
   value: -15
});
